# muapi integrations

> Integrate with muapi using LangChain Python.

[muapi](https://muapi.ai) is a generative-media API that aggregates 250+
image, video, audio, edit, enhance, lipsync, and 3D models behind a single
key. Power your agentic media workflows with:

* A unified **submit → poll** API across every model
* A built-in **model selector** that ranks the right model for a brief
* **40+ named skills** — pre-baked multi-step recipes (UGC ads, storyboards,
  product videos, social packs, etc.)
* A **creative agent** that decomposes briefs into DAGs and executes them

## Installation and setup

* Sign up at [muapi.ai](https://muapi.ai) to get an API key, then set it in
  the environment:

  ```bash
  export MUAPI_API_KEY="..."
  ```

  Or store it in the OS keychain via the official CLI: `muapi auth configure`.

* Install the integration:

  ```bash
  pip install muapi-langchain
  ```

For the Deep Agents recipe below, also install the `deepagents` extras:

```bash
pip install "muapi-langchain[deepagents]"
```

## Document loader

Load previously-generated muapi assets as LangChain `Document`s — handy for
RAG over a user's generation history or for "do another in that style".

```python
from muapi_langchain import MuapiAssetLoader

loader = MuapiAssetLoader(request_ids=["req_abc", "req_def"])
docs = loader.load()
for d in docs:
    print(d.metadata["url"], "·", d.page_content[:80])
```

## Deep Agents integration

The integration ships **4 tools** that form a capability gradient:

| Tool                    | Description                                                | When to use                                  |
| ----------------------- | ---------------------------------------------------------- | -------------------------------------------- |
| `muapi_select`          | Rank models + skills for an intent. No credits spent.      | Discovery — always try this first            |
| `muapi_generate`        | Single-shot generation (image / video / audio / edit)      | One clear ask, one asset out                 |
| `muapi_run_skill`       | Named multi-step recipe (UGC ad, storyboard, product video) | The user's ask matches a known workflow      |
| `muapi_creative_agent`  | Open-ended planner — decomposes brief into DAG, executes   | Multi-asset / multi-modal briefs             |

The recommended pattern splits these across a planner and a
`creative-specialist` subagent: cheap tools (`select`, `generate`) on the
planner, expensive tools (`run_skill`, `creative_agent`) on the specialist,
with `interrupt_on` gating open-ended creative work for human approval.

### Environment variables

```bash
export MUAPI_API_KEY="..."
export OPENAI_API_KEY="..."
```

### Define the tools

The tools are already `@tool`-decorated — just import them:

```python
from muapi_langchain import (
    muapi_select,
    muapi_generate,
    muapi_run_skill,
    muapi_creative_agent,
    PLANNER_TOOLS,        # [muapi_select, muapi_generate]
    SPECIALIST_TOOLS,     # [muapi_run_skill, muapi_creative_agent]
)
```

### Build the agent

```python
import os, uuid
from deepagents import create_deep_agent
from langchain_openai import ChatOpenAI
from langgraph.checkpoint.memory import MemorySaver
from muapi_langchain import (
    MuapiCostCallback, PLANNER_TOOLS, SPECIALIST_TOOLS,
)

CREATIVE_SPECIALIST = {
    "name": "creative-specialist",
    "description": (
        "Handles multi-step muapi workflows: named skills (UGC ads, "
        "storyboards, product videos) and open-ended creative briefs."
    ),
    "system_prompt": (
        "You are a muapi creative specialist. "
        "Prefer muapi_run_skill when the user's brief matches a named recipe. "
        "Escalate to muapi_creative_agent only for open-ended multi-asset briefs."
    ),
    "tools": SPECIALIST_TOOLS,
}

agent = create_deep_agent(
    model=ChatOpenAI(model="gpt-4o", api_key=os.environ["OPENAI_API_KEY"]),
    tools=PLANNER_TOOLS,
    subagents=[CREATIVE_SPECIALIST],
    system_prompt=(
        "You are a creative-media Deep Agent backed by muapi. "
        "Always start with muapi_select to discover which models/skills "
        "fit the brief. For a single image/video/audio ask, call "
        "muapi_generate yourself. Delegate multi-step asks to the "
        "creative-specialist subagent."
    ),
    interrupt_on={
        "muapi_creative_agent": {"allowed_decisions": ["approve", "edit", "reject"]},
    },
    checkpointer=MemorySaver(),
)
```

### Track spend with `MuapiCostCallback`

```python
cost_cb = MuapiCostCallback(
    budget_credits=500,
    on_event=lambda evt, payload: print(f"[{evt}] {payload}"),
)

config = {
    "configurable": {"thread_id": str(uuid.uuid4())},
    "callbacks": [cost_cb],
}

result = agent.invoke(
    {"messages": [{"role": "user", "content":
        "Make a 3-shot Instagram carousel for SunFizz mango sparkling water."
    }]},
    config=config,
)

print(cost_cb.summary())
```

If `budget_credits` is exceeded mid-run, the callback raises `BudgetExceeded`
to abort the agent cleanly.

### Run with human-in-the-loop approval

`interrupt_on` pauses execution before any `muapi_creative_agent` call so
you can review the proposed plan before credits are spent.

```python
from langgraph.types import Command

while getattr(result, "interrupts", None):
    action = result.interrupts[0].value["action_requests"][0]
    print(f"Pending: {action['name']}\nArgs: {action['args']}")
    decision = input("approve / edit / reject: ").strip().lower()
    result = agent.invoke(
        Command(resume={"decisions": [{"type": decision}]}),
        config=config,
        version="v2",
    )
```

### Decision tree: which tool to use

```
User brief
  └─ Don't know which model/skill? ───→ muapi_select   (free, planner)
  └─ Single asset, clear prompt? ─────→ muapi_generate (planner)
  └─ Matches a named workflow?
       (UGC ad, storyboard, product video, social pack, …)
                              ───────→ muapi_run_skill (specialist)
  └─ Multi-asset / multi-modal brief? ─→ muapi_creative_agent
                                          (specialist, interrupt_on)
```
